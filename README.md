# Crypto Tracker
 A OneStop application to track all the crypto currency changes in a single mobile app.
## Key Features
- A customized Line Chart(using Canvas) graph for tracking the coin prices over past 5 days.
- An indepth coin detail about the latest price change based on market.
- A list of all the in - market Crypto Coins.
- Type-safe navigation with Kotlinx Serialization.


# Tech Stack
- Jetpack Compose
- Kotlin
- Koin
- Ktor
- Canvas
